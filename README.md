### How to run

1. `create a `.env` in app directory with the following secret ...

```sh
  CONSUMER_KEY='###########'
  CONSUMER_SECRET='###########'
  ACCESS_TOKEN_KEY='###########'
  ACCESS_TOKEN_SECRET='###########'
```
2. type `npm run dev` to start application in terminal.
